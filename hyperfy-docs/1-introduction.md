# Introduction

Hyperfy is a powerful platform for creating and hosting persistent 3D virtual worlds with real-time interactions. This documentation will guide you through understanding and using the Hyperfy platform.

## Overview

Hyperfy enables you to create standalone persistent worlds that can be hosted on your own domain. It provides a robust framework for real-time interactions, content creation, and development, making it suitable for various applications from social spaces to interactive experiences.

## Features

- **Standalone Persistent World**: Create and maintain persistent 3D environments
- **Custom Domain Hosting**: Host worlds on your own domain
- **Portable Avatars**: Connect via Hyperfy for portable avatar support
- **Real-time Content Creation**: Create and modify content directly in-world
- **In-world Development**: Real-time coding capabilities for developers
- **Interactive Apps**: Fully interactive and interoperable app format
- **Extensible Architecture**: Highly extensible system for custom functionality

## Use Cases

Hyperfy can be utilized for various applications, including:

- **Live Events**: Host virtual concerts, conferences, and gatherings
- **Storefronts**: Create interactive virtual shopping experiences
- **Podcasts**: Host immersive podcast recordings and listening sessions
- **Gaming**: Develop interactive gaming experiences
- **Social Spaces**: Build community spaces for social interaction

## Getting Started

### Prerequisites

Before you begin working with Hyperfy, ensure you have:

- Node.js version 22.11.0 or higher (recommended to use nvm for Node.js version management)
- Basic understanding of JavaScript/Node.js
- Git installed on your system

### Installation

1. Clone the repository:
```bash
git clone https://github.com/hyperfy-xyz/hyperfy.git my-world
```

2. Navigate to the project directory:
```bash
cd my-world
```

3. Set up environment variables:
```bash
cp .env.example .env
```

4. Install dependencies:
```bash
npm install
```

5. Start the development server:
```bash
npm run dev
```

### Quick Start Guide

Once you have completed the installation:

1. The development server will start and provide you with a local URL
2. Access the URL in your web browser to enter your world
3. Begin exploring the basic features and interactions
4. Refer to the subsequent sections of this documentation for detailed information about specific features and development guides

Note: Hyperfy is currently in alpha phase as it transitions from the reference platform to fully self-hostable worlds. While most features are available, APIs may change during this transition period.
